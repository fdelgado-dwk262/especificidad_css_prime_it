¿De qué color se mostrará finalmente el texto del párrafo?

Calcula la especificidad de cada regla que afecta al párrafo.

Calcula la especificidad de cada regla que afecta al párrafo. ¿Por qué la regla 4 se impone sobre las demás, incluso si alguna pudiera tener más especificidad?

Si elimináramos !important de la regla 4, ¿cuál sería el color final? Justifica.

        

          /* Regla 1 */
          p {
            color: black;
          }
          
          /* Regla 2 */
          .modulo p.destacado {
            color: orange;
          }
          
          /* Regla 3 */
          #contenedor .destacado {
            color: blue;
          }
          
          /* Regla 4 */
          #contenedor p.texto.destacado {
            color: green !important;
          }
          
          /* Regla 5 */
          section.modulo #contenedor .texto {
            color: red;
          }
<section class="modulo"> <div id="contenedor"> <p class="texto destacado">Texto importante</p> </div> </section>

Texto importante

Resultado final: La regla 4 gana porque incluye !important, que anula todas las demás, incluso si alguna tiene igual o mayor especificidad.

El color final del texto es: green

¿Qué pasa si eliminamos !important? Ahora se comparan solo por especificidad.

La regla 4 y la regla 5 tienen igual especificidad (0,1,2,1).

Como la regla 5 aparece después en la hoja de estilos, predomina por orden en la cascada.

Color final: red
